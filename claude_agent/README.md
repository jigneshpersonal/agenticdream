# Claude Agent Structure

This folder contains the scaffold for a Claude-based agent setup.

## Folders

- `config/`: Claude model configuration, credentials, and runtime settings.
- `prompts/`: Reusable prompt templates and instruction files.
- `tools/`: Integration tool definitions, adapters, and helper wrappers.
- `data/`: Reference data, examples, or asset files used by the agent.
- `src/`: Agent implementation code and orchestration logic.
- `tests/`: Unit and integration tests for the Claude agent.
