# agenticdream
A collection of Supreme Agents to get things done

## Claude Agent Folder Structure

This repository now includes a scaffold for a Claude-based agent implementation.

- `claude_agent/config/` — model settings, credentials, and runtime configuration.
- `claude_agent/prompts/` — prompt templates and instruction documents.
- `claude_agent/tools/` — tool integrations and adapters for Claude.
- `claude_agent/data/` — example data, references, and assets.
- `claude_agent/src/` — core agent implementation code.
- `claude_agent/tests/` — tests for the Claude agent.
